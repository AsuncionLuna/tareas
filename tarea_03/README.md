## Proceso de la tarea: 

* Realicé la visualización en formato de infografía sobre la identidad sonora del rock chileno.
* Organicé la información en una estructura narrativa para una página web estática.
* Utilicé etiquetas HTML para incorporar títulos, párrafos, imágenes, listas y enlaces.
* Integré la crónica desarrollada para la entrega anterior junto a los gráficos.
* Incorporé tres imágenes: la infografía general, el gráfico radar y el gráfico de barras.
* Probé la visualización en el navegador para verificar que las imágenes y enlaces funcionaran correctamente.
* Preparé los archivos para su publicación mediante GitHub Pages.


El objetivo fue mostrar si existe una identidad sonora compartida dentro del rock chileno, además de comunicar los principales hallazgos obtenidos a partir del análisis de las 250 mejores canciones del rock chileno.
